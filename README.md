## 项目介绍
基于SSM框架的网上书城系统，有用户和管理员两个角色，主要功能如下
用户：图书分类浏览、书籍查询、书籍购买、购物车、订单信息
管理员：图书管理、分类管理、订单管理等

## 项目技术
- 编程语言：Java
- 数据库：MySQL
- 前端技术：JSP、JavaScript、jquery
- 后端技术：Spring、SpringMVC、MyBatis

## 运行环境
- JDK版本：JDK1.8及以上
- 开发工具：IDEA、Ecplise、Myecplise都可以
- 数据库: MySQL5.7及以上
